How Can I Change Data inside The Theme -------------------------------

First, Thank you for getting our Theme -------------------------------
We are glade that you are using it -----------------------------------

----------------------------------------------------------------------
----------------------------------------------------------------------

HEADER DATA -----------------------------------------------------Start

Where you can find Header data [Site Title, Site Name, Site Description, Login Data, Register Data]:
Go to folder (includes -> templates -> header.php)

To Change Site Title That appear in the browsers from the top: click (CTRL + f) and type (#Title) search.

To Change Site Name: click (CTRL + f) and type (#Name) search.

To Change Site Description: click (CTRL + f) and type (#Description) search.

To Change Login Data: click (CTRL + f) and type (#LoginData) search.

To Change Register Data: click (CTRL + f) and type (#RegisterData) search.

HEADER DATA -------------------------------------------------------End

----------------------------------------------------------------------
----------------------------------------------------------------------

MENU DATA -------------------------------------------------------Start

Where you can find Menu bar data [Cart, About Us, Our Location, ...]:
Go to folder (includes -> templates -> menubar.php)

You can change data easily.

MENU DATA ---------------------------------------------------------End

----------------------------------------------------------------------
----------------------------------------------------------------------

FOOTER DATA -----------------------------------------------------Start

Where you find Footer data [copy right, facebook link, social media links]:
Go to folder (includes -> templates -> footer.php)

To Change Copy right: click (CTRL + f) and type (#Copyright) search. 

To Add Facebook link: click (CTRL + f) and type (#FacebookLinkHere) search.

To Add Twitter link: click (CTRL + f) and type (#TwitterLinkHere) search.

To Add Instagram link: click (CTRL + f) and type (#InstagramLinkHere) search.

To Add Youtube link: click (CTRL + f) and type (#YoutubeLinkHere) search.

Code To Add Another Site link and icon:
You will add this part:

	<a class="mr-2" href="YourSiteLinkHere"> 
		<i class="fab fa-youriconnamefromfontawesome.com"></i>
	</a>
	
for more description see footer page.

FOOTER DATA -------------------------------------------------------End
